Changelog

A new version of the H.265 / HEVC Player has been installed. Please see below for a list of recent changes.

Further information are available at libde265.org.

0.2.2

  • Fixed error when opening files on newer versions of Chrome.

0.2.1

  • Support Main10 streams.
  • HEVC version 2 RExt support (excluding 16 bit and high throughput profiles).
  • VUI decoding.

0.2.0

  • Updated to libde265 1.0.0.
  • Added subtitle support (SSA/ASS format) if contained in video file.
  • Handle case where audio frame layout changes in the stream.
  • Started writing public changelog.
